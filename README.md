# Smart Todo List

## Краткое описание
Smart Todo List — это мобильное приложение для управления задачами с категориями и приоритетами. 
Помогает эффективно организовать ежедневные дела и получать советы по продуктивности.

## Основные функции
1.  **Добавление задач:** Пользователь создает задачи с названием, категорией, приоритетом и необязательным описанием.
2.  **Просмотр задач:** Все задачи отображаются в виде списка с группировкой по категориям.
3.  **Хранение данных:** Все задачи сохраняются локально на устройстве.
4.  **Советы по продуктивности:** На главном экране отображается случайный совет по продуктивности, загружаемый из публичного API.

## Структура экранов

### Экран 1: Главный экран (Home Screen)
*   **Заголовок:** "Мои задачи"
*   **Виджет совета:** Отображает случайный совет по продуктивности.
*   **Список категорий:** Группировка задач по категориям (работа, дом, личное, покупки).
*   **Элементы задач:** Для каждой задачи отображается название, категория, приоритет и статус выполнения.
*   **Кнопка "Добавить задачу":** Открывает экран создания новой задачи.
*   **Кнопка "Статистика":** Переход на экран статистики.

### Экран 2: Экран добавления задачи (Add Task Screen)
*   **Заголовок:** "Новая задача"
*   **Поле названия:** Поле для ввода названия задачи.
*   **Выбор категории:** Выпадающий список с категориями.
*   **Выбор приоритета:** Переключатель между высоким/средним/низким приоритетом.
*   **Поле описания:** Необязательное поле для подробного описания.
*   **Кнопка "Сохранить":** Сохраняет новую задачу.
*   **Кнопка "Назад":** Возврат на главный экран.

### Экран 3: Экран статистики (Statistics Screen)
*   **Заголовок:** "Статистика"
*   **Общая статистика:** Количество всех задач и процент выполнения.
*   **Статистика по категориям:** Прогресс выполнения для каждой категории.
*   **Кнопка "Назад":** Возврат на главный экран.

## Навигация
Главный экран (Home Screen) <--> Экран добавления задачи (Add Task Screen)
Главный экран (Home Screen) <--> Экран статистики (Statistics Screen)

Навигация реализуется с помощью простого `Navigator` в Flutter. 
Кнопка "Добавить задачу" на главном экране осуществляет переход на экран добавления. 
Кнопка "Статистика" открывает экран статистики. Кнопки "Назад" (или стрелки в AppBar) возвращают пользователя на главный экран.
